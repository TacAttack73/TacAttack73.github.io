Class: numerical_results

numerical_util.numerical_results(result, completed)

Stores the result of a numerical method and whether the numerical method completed successfully or was interrupted mid-way.

Constructor

new numerical_results(result, completed)

Parameters:
Name Type Description
result any Result of numerical method.
completed boolean Whether numerical method completed successfully or was interrupted mid-way, likely due to a given maximum iteration count being exceeded. Default value is true.

toString() → {string}

Converts this numerical_results to a string. String format rules:
  • Starts with left brace.
  • String "result = " comes next.
  • Result comes next.
  • String ", max_iterations = " comes next.
  • Completed comes next.
  • Ends with right brace.
Examples include:
  • {result = 1.414, completed = true}
  • {result = -1, completed = false}
