Class: continuation_options

numerical_util.continuation_options(error_tolerance, max_iterations)

Contains an error tolerance and maximum iteration count that gets supplied to iterative numerical methods.

Constructor

new continuation_options(error_tolerance, max_iterations)

Parameters:
Name Type Description
error_tolerance number The error tolerance of the iterative numerical method. Iteration generally stops when the norm of the difference of the current & previous iterations is less than error tolerance.
max_iterations number The maximum iteration count before the iterative numerical method is forced to abort. This ensures that infinite loops do not occur. Set to Number.MAX_SAFE_INTEGER by default.

Methods

toString() → {string}

Converts this continuation_options to a string. String format rules:
  • Starts with left brace.
  • String "error_tolerance = " comes next.
  • Error tolerance comes next.
  • String ", max_iterations = " comes next.
  • Max iterations comes next.
  • Ends with right brace.
Examples include:
  • {error_tolerance = 0.005, max_iterations = 12}
  • {error_tolerance = 0.5, max_iterations = 1024}
